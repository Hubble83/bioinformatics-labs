Análise de Homologias por Blast


A ferramenta Blast é normalmente utilizada para fazer uma análise de homologias. Desta forma, foi desenvolvido scripts com o objetivo de correr o blast para todos estes genes, de forma a poder encontrar possíveis funções para aqueles cuja função é desconhecida.

Este processo cria dois ficheiro BlastCSV.csv e tabela_com_seq_Completa.csv, que são depois associados para criar os ficheiros que interessam, com as informações de cada gene.

Todos os ficheiros estão disponíveis no GitHub, e é possível aceder a esse sítio através de "Scripts Desenvolvidos" neste website.
