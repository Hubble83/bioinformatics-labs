Análise da sequência

Extracção e análise


Inicialmente, é pretendido aceder ao NCBI e guardar o genoma num ficheiro genbank no intervalo pretendido, entre os genes referidos como lpg2079 e 2310 (2327101-2610800). Para tal, esta extração é realizada através do script SeqRecord.py. Este script guarda automaticamente no ficheiro seq_record.gb a informaçao relativa ao genoma. Uma vez feito este processo, como output são escritas algumas informações sobre este organismo, que foram guardados em ProteinData.txt, características como o seu ID, nome comum, taxonomia, entre outros. A sequência completa referente a esse ficheiro é escrita no ficheiro sequence.txt.


Validação


Para validar esta extração, é realizada uma comparação do nosso ficheiro com a tabela indicada pelo professor (ProteinTable1.txt). Esta análise está presente no ficheiro analise.txt e é também ela produto do output deste script (Validation.py).

Geração de tabelas


Recorrendo ao ficheiro seq_record.gb, através da execução do ficheiro script.py, são gerados dois ficheiros com as informações dos vários genes e proteínas correspondentes: o table.csv para abrir a tabela localmente, e o table.html contendo as várias linhas com a formatação necessária para serem inseridas na tabela resumo.
